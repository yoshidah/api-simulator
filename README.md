api-simulator
===

Overview
This api-simulator is a server system that acts as a simulator for testing purposes that communicates with clients such as smartphones using JSON API.

## Description

## Requirement
Node.js v6.11.0 or above

## Install
`
$ npm install
`

## Usage
`
$ node api-simulator.js
`
